# Presupuesto Acapulco — E9 y E10

Referencia para el proceso PRESUPUESTO en Acapulco: dos presupuestos (plantas lógicas) **E9** y **E10**. Al usar el comando de presupuesto, el primer paso debe ser: **¿Planta: E9 o E10?**

---

## 1. Flujo de entrada

1. Usuario envía comando de presupuesto (ej. `presupuesto` o `solicitar presupuesto`).
2. **Paso 0 (Acapulco):** Si la empresa/contexto es Acapulco → preguntar: **«¿Planta? 1) E9  2) E10»**. La respuesta define el presupuesto (E9 o E10) para el resto del flujo.
3. Paso 1: Categoría (las 7 categorías son las mismas para E9 y E10).
4. Paso 2: Subcategoría (según la categoría elegida y la planta E9 o E10; listas abajo).
5. Resto del flujo: monto, concepto, PDF obligatorio, confirmación.

---

## 2. Las 7 categorías (comunes a E9 y E10)

| # | Categoría           |
|---|---------------------|
| 1 | NOMINA              |
| 2 | RENTAS              |
| 3 | SERVICIOS           |
| 4 | TALLER              |
| 5 | MANTENIMIENTO       |
| 6 | GASTOS GENERALES    |
| 7 | IMPUESTOS PLANTA    |

*(Las filas "TOTAL ..." en Excel son sumas y no deben ofrecerse como opción de subcategoría.)*

## 5. Uso en base de datos

- **presupuesto_catalogo:** Una fila por (planta_id, categoria, subcategoria, activo).  
  - En `plantas`: tener dos registros para Acapulco, ej. nombre `E9` y `E10` (o "Acapulco E9" / "Acapulco E10") para que `planta_id` identifique E9 vs E10.
- **presupuesto_asignaciones:**  
  - Si el saldo es **un total por planta/periodo:** una fila por (planta_id, periodo) con `monto_aprobado` = suma de todas las subcategorías de esa tabla para esa planta.  
  - Si el saldo es **por subcategoría:** una fila por (planta_id, periodo, categoria, subcategoria, monto_aprobado) usando las tablas de arriba; el saldo disponible por subcategoría sería ese monto menos la suma de solicitudes aprobadas de esa misma (planta, periodo, categoria, subcategoria).

Este documento sirve como fuente única para cargar catálogo y asignaciones (Excel/VBA o scripts) y para implementar el menú "Planta E9/E10 → Categoría → Subcategoría" en el bot.
